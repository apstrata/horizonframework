Create folders:
	<projectFolder>/lib
	<projectFolder>/lib/apstrataSDK
	<projectFolder>/lib/dojo
	
import latest dojo SDK into lib/dojo, the folder will look like:
	<projectFolder>/lib/dojo/dojo
	<projectFolder>/lib/dojo/dijit
	<projectFolder>/lib/dojo/dojox
	<projectFolder>/lib/dojo/util

import latest apstrata SDK into lib/apstrataSDK, the folder will look like:
	<projectFolder>/lib/apstrata
